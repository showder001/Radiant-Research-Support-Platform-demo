# Radiant 高保真原型设计文档

## 概述

这是一个完整的 Radiant 学术研究协作工具的交互式高保真原型，基于您提供的设计指导方案实现。该原型使用 React + TypeScript + Tailwind CSS 构建，可以直接在浏览器中运行和交互。

## 设计实现

### 一、核心设计原则

#### 1. 清晰与专业
- 使用深蓝色 (#0F3B6C) 和科技紫色 (#4A2E9E) 作为主色调
- 采用无衬线字体，层级清晰
- 浅灰色背景 (#F8F9FA) 提供舒适的阅读环境
- 适度的阴影和圆角营造现代感

#### 2. 知识可视
- 知识图谱使用不同颜色区分节点类型：
  - 文献：蓝色 (#4A90E2)
  - 人员：绿色 (#50C878)
  - 机构：橙色 (#E27A4A)
  - 代码：黄色 (#F39C12)
- 实时力导向图可视化
- 小型网络图摘要展示关键节点动态

#### 3. 流畅协作
- 右侧智能侧边栏提供统一的 AI 协作体验
- 根据上下文动态显示相关操作
- 无缝的多功能切换

### 二、核心界面实现

#### 1. 仪表盘 (Dashboard)
**功能特点：**
- 自然语言搜索框，支持智能提问
- 近期动态卡片流，实时显示知识图谱更新
- 研究趋势可视化，展示热门领域增长趋势
- 常用子图快速访问
- SVG 绘制的小型知识图谱概览

**交互设计：**
- 搜索框点击触发智能调研
- 动态卡片悬停高亮
- 标签点击快速筛选

#### 2. 知识图谱浏览器 (KG Explorer)
**功能特点：**
- 基于 react-force-graph-2d 的交互式图谱
- 节点筛选和时间范围控制
- 布局算法切换（力导向、圆形、径向）
- 节点详情侧边面板
- 缩放和平移控制

**交互设计：**
- 节点点击查看详情
- 悬停显示标签
- 拖拽调整布局
- 动态连线展示关系

#### 3. 论文工作台 (Paper Workshop)
**功能特点：**
- 三栏布局：文档导航 + 编辑器 + 智能审稿
- Markdown 编辑器支持 LaTeX
- 多审稿人模拟系统
- 审稿意见一键应用
- 文献引用管理

**交互设计：**
- 审稿人卡片切换
- 意见标记与应用
- 文档大纲导航
- 实时预览功能

#### 4. 智能侧边栏 (Intelligent Sidebar)
**功能特点：**
- 显示当前 Agent 身份
- 知识来源标注
- 建议操作按钮
- 对话历史记录
- 上下文感知提示

**交互设计：**
- 流式对话界面
- 可点击的知识来源
- 智能建议快速触发
- 自然语言输入

#### 5. 项目管理 (Projects)
**功能特点：**
- 项目卡片展示
- 进度追踪
- 成员和文献统计
- 状态管理

#### 6. 收藏夹 (Favorites)
**功能特点：**
- 分类标签页：文献、作者、机构、代码
- 搜索和筛选
- 星标管理
- 详细信息展示

#### 7. 用户流程演示 (User Flow Demo)
**功能特点：**
- 两个典型场景演示
- 步骤进度指示器
- 交互式演示流程
- 详细说明文档

### 三、典型用户流程

#### 场景一：调研新方向
1. **仪表盘搜索** → 输入自然语言问题
2. **跳转知识图谱** → 自动切换视图
3. **图谱展开** → 相关节点高亮
4. **智能总结** → AI 助手生成文本
5. **建议操作** → 提供下一步选项

#### 场景二：论文打磨
1. **打开论文** → 加载编辑器
2. **启动审稿** → 点击模拟审稿
3. **选择审稿人** → 查看审稿人画像
4. **查看意见** → 生成详细反馈
5. **应用修改** → 一键插入建议

### 四、技术实现

#### 技术栈
- **框架：** React 18 + TypeScript
- **样式：** Tailwind CSS v4
- **UI 组件：** Radix UI
- **图谱可视化：** react-force-graph-2d
- **图标：** Lucide React
- **构建工具：** Vite

#### 关键组件
```
src/app/components/
├── Dashboard.tsx              # 仪表盘
├── KnowledgeGraphViewer.tsx   # 知识图谱浏览器
├── PaperWorkshop.tsx          # 论文工作台
├── IntelligentSidebar.tsx     # 智能侧边栏
├── ProjectsPage.tsx           # 项目管理
├── FavoritesPage.tsx          # 收藏夹
└── UserFlowDemo.tsx           # 用户流程演示
```

### 六、使用指南

#### 浏览功能
1. **仪表盘** - 查看研究动态和趋势
2. **知识图谱** - 探索节点和关系
3. **论文工作台** - 编辑和审稿
4. **项目** - 管理研究项目
5. **收藏夹** - 整理重要资源
6. **用户流程** - 查看交互演示

#### 交互操作
- **点击节点** - 查看详细信息
- **拖拽图谱** - 调整视图
- **切换审稿人** - 查看不同意见
- **点击建议** - 快速执行操作
- **搜索内容** - 智能筛选

### 七、设计延展

## 总结

这个高保真原型完整实现了 Radiant 的核心功能和交互设计，可以作为：
- **产品演示** - 向投资人和用户展示
- **用户测试** - 收集反馈和改进建议
- **开发参考** - 指导后续开发工作
- **设计规范** - 建立统一的视觉语言

原型充分展现了 Radiant 作为学术研究协作工具的独特价值：将知识图谱可视化、AI 智能协作和多功能工作台无缝整合，为研究人员提供一体化的研究体验。
